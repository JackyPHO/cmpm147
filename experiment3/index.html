Experiment 03: Alternate Worlds

Description

Creating a dungeons and overworld level generator using autotiling logic

Technical

I am messing around with the generateGrid functions to set some characters to place certain tiles on a particular character. I mainly took inspiration from the Rouge example. Made a dungeons and overworld button so that the user can switch between each scene. I brought life to the project by adding random chests inside each room in the dungeon then clicking on the chest to make them disappear. Made a dungeons and overworld button so that the user can switch between each scene.
